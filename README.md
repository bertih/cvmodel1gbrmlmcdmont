# 📄 Currículo Profissional Interativo | Gabriel M. Monteiro
Meu projeto de currículo interativo, básico.

Este projeto, desenvolvido para apresentar a minha trajetória como **Administrador Esp. em Gestão de Crédito, Cobrança, Controladoria e Finanças** de uma forma moderna, interativa e de fácil acesso para recrutadores.

##  O Projeto

O objetivo deste repositório é alojar uma versão web do meu currículo, garantindo que o meu perfil profissional esteja sempre disponível online. O design foi estrategicamente criado para ser legível, elegante e otimizado para sistemas de recrutamento (ATS - *Applicant Tracking Systems*).

###  Principais Funcionalidades

* ** Design Corporativo e Moderno:** Paleta de cores em tons de azul marinho (*Navy Blue*), transmitindo profissionalismo e confiança(Meu tom favorito).
* ** Alternador de Tema:** Suporte para Modo Claro (*Light Mode*) e Modo Escuro (*Dark Mode*), permitindo uma leitura confortável em qualquer ambiente de iluminação. (Ainda tem gente que utiliza o branco. O branco queima meus olhos).
* ** Totalmente Responsivo:** Layout que se adapta perfeitamente a ecrãs de computadores, tablets e celulares.
* ** Interatividade Dinâmica:** Cartões de experiência profissional expansíveis, mantendo a interface limpa e organizada.
* ** Otimizado para ATS:** Estrutura semântica de código limpa, facilitando a extração de palavras-chave por Inteligências Artificiais de recrutamento.

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização avançada, uso de variáveis para gestão dos temas, transições suaves e *media queries* para design responsivo.
* **JavaScript (Vanilla):** Lógica simples, leve e sem dependências, focada apenas na alternância de temas e interatividade.
